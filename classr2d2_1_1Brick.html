Detailed Description

An unconfigured NXT brick, without any sensors or motors attached.

An unconfigured NXT brick can issue system commands such as Brick::getFirmware or Brick::getDeviceInfo, but it doesn't have any sensors or motors attached, instead use Brick::configure to setup an NXT layout.
